API Project: Request Header Parser Microservice

  1. The API endpoint is GET [url]/api/whoami
  2. The end point will return the json object with ip address and headers with ['accept-language'] and ['user-agent'].

Example Usage:

  • [url]/api/whoami

Example Output:

{"ip":"::ffff:127.0.0.1","language":"en-US,en;q=0.5","software":"Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:95.0) Gecko/20100101 Firefox/95.0"}
